## 安装

### 前提条件

- Python 3.11

### 步骤

1. **克隆仓库**：

   ```sh
   git clone https://github.com/lalifeier/proxy-scraper.git
   cd proxy-scraper
   ```

1. **安装依赖**：
   ```sh
   pip install -r requirements.txt
   ```
## 使用方法

### 运行脚本

可以通过命令行参数运行脚本以进行爬取或验证。

#### 爬取代理
 ```sh
python main.py --scrape
 ```

#### 验证代理

 ```sh
python main.py --validate
 ```
